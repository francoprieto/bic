🔐 Prueba de Protocolo bic://

Nota: Para que estos enlaces funcionen, debe tener instalada la aplicación Bic y el protocolo bic:// debe estar registrado en su sistema.

Pruebas Básicas

1. Prueba Simple

Abrir Bic (Prueba Simple)
bic://test

2. Con Archivos Locales

Firmar Archivo Local
bic://?files=file:///home/usuario/documento.pdf

3. Múltiples Archivos

Firmar Múltiples Archivos
bic://?files=file:///home/usuario/doc1.pdf,file:///home/usuario/doc2.pdf

Pruebas con Archivos Remotos

4. Archivo Remoto Simple

Firmar Archivo Remoto
bic://?files=https://ejemplo.com/documento.pdf

Instrucciones de Instalación

Si instaló con .deb:

El protocolo debería registrarse automáticamente durante la instalación.

Si usa AppImage o instalación manual:

  1. Ejecute el script de registro:
    ./register-protocol.sh
  2. Reinicie su navegador
  3. Pruebe los enlaces en esta página

Verificar el registro:

xdg-mime query default x-scheme-handler/bic

Debería mostrar: bic-electron.desktop

Solución de Problemas

El navegador no reconoce el protocolo:

  • Verifique que la aplicación esté instalada
  • Ejecute el script de registro manualmente
  • Reinicie el navegador completamente
  • En algunos navegadores, puede necesitar dar permiso la primera vez

Verificar manualmente:

xdg-open 'bic://test'

Este comando debería abrir la aplicación Bic

Navegadores soportados:
  • Firefox
  • Chrome/Chromium
  • Edge
  • Otros navegadores basados en Chromium
